LiveWeather is a simple and responsive weather application built using **React.js** that fetches real-time weather information using the **OpenWeatherMap API**. Users can search any city and instantly view its current weather details.

## Features

- Search weather by entering any city name
- Displays real-time weather data:
  - Temperature
  - Minimum Temperature
  - Maximum Temperature
  - Humidity
  - Feels Like Temperature
  - Weather Description
- Dynamic weather image changes based on conditions
- Clean and responsive UI
- Built using Material UI components
- Hosted and deployed on Render

---

## Tech Stack Used

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Material UI
- OpenWeatherMap API
- Git & GitHub
- Render (Deployment)

---

## How It Works

1. User enters the city name in the search box.
2. Application sends request to OpenWeatherMap API.
3. API returns real-time weather data.
4. Weather details are displayed dynamically inside the InfoBox.
